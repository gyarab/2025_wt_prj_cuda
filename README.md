# 2025_wt_prj_cuda
Tento projekt bude webová databáze obchodů (trading journal) vytvořená v Djangu. Uživatel si bude moci ukládat své obchody na finančních trzích a sledovat jejich úspěšnost. Každý obchod bude obsahovat informace jako aktivum (např. akcie nebo kryptoměna), typ obchodu (long / short), vstupní a výstupní cena, datum, velikost pozice a zisk nebo ztrátu. Systém umožní filtrování obchodů podle strategie nebo trhu a zobrazí statistiky úspěšnosti. Cílem projektu je vytvořit přehledný nástroj pro analýzu trading výkonu a zlepšování obchodních strategií pomocí databázového systému.
